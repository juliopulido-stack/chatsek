Recuperación de Acceso

Utiliza esta herramienta para crear un nuevo usuario de emergencia y recuperar el acceso al sistema.

Pasos siguientes:
  1. Crea el usuario con este formulario.
  2. Accede a tu Firebase Console.
  3. Busca la colección users en Firestore.
  4. Localiza el documento del nuevo usuario (por email).
  5. Cambia el campo role de "usuario" a "super_admin".
  6. Ya podrás entrar como administrador desde la página principal.
